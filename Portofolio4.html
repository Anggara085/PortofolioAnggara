About My Project

Proyek ini membangun Sistem Monitoring Kesehatan Real-Time menggunakan IoT, ESP32, dan sensor DHT22 untuk memantau suhu dan kelembapan tubuh melalui Wi-Fi.

Tujuan proyek ini adalah memberikan data kesehatan real-time yang mudah diakses oleh pengguna dan tenaga medis untuk meningkatkan pemantauan kondisi kesehatan.

Teknologi IoT dan aplikasi mobile dapat mempermudah pengelolaan kesehatan, menjadikan proyek ini langkah awal menuju solusi kesehatan berbasis teknologi.
